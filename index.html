💾 Make Logins Permanent

Your account list is already saved in your browser. The logins vanish because Discord's cookies only survive if you keep the launcher at a fixed URL. Pick one option below:

Option A — Bookmark this file Easiest

Save this HTML file somewhere permanent (e.g. Desktop). Bookmark it in your browser — don't move or rename the file. Open via the bookmark every time so the file path never changes. Your account list and Discord cookies will persist.

Option B — Serve it locally Best

Run a tiny local server so the URL is always http://localhost:8080. Open Terminal / Command Prompt in the folder with this file and run: 

python3 -m http.server 8080

Or if you have Node:  npx serve . -p 8080

Then open http://localhost:8080/discord-launcher.html and bookmark that. Keep the terminal running in the background.

Why logins stay after this

Discord stores your login in a browser cookie on discord.com / ptb.discord.com / canary.discord.com. Those cookies last weeks and survive restarts — as long as you don't clear browser data. Your account nicknames are in localStorage tied to this file's URL, so a fixed URL = permanent saves.
